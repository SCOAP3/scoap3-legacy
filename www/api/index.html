All articles published Open Access through the SCOAP3 initiative, with complete metadata and several download formats are publicly available on the SCOAP3 repository, in addition to the platforms of participating publishers. Public OAI-PMH feeds are also available.

The SCOAP3 API offers a pilot service to SCOAP3 partners to search and download articles and the corresponding metadata. A full documentation of the API is available here.

The use of the API requires a token which will be provided upon registration.
